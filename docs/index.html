Cave View

... a 3D cave survey viewer for the web, version 2.7.0

Features

Displays:

  • Survex .3d
  • Therion .lox models
  • Compass .plt models (basic support tested)
  • Terrain and "Slippy Map" overlays

No plugins required.

Demos

  • Multiple instances
  • DTM and Cesium terrain
  • Using "entrance" events
