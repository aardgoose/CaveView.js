Cave View

a 3D cave survey viewer for the web

Features

Displays:

  • Survex .3d
  • Therion .lox models
  • Terrain and "Slippy Map" overlays

No plugins required.

Tested with the following browsers:

  • Google Chrome 68
  • Firefox 61
  • Internet Explorer 11
  • Microsoft Edge
  • Opera 54
  • Safari 9.1 (on iOS and OS X)
